# Study Notes

## 목적
CKArena를 AI agent와 함께 개발하면서 더 깊게 공부해야 할 개념을 모아둔다.

## 공부할 주제
- React 프로젝트 구조.
- Spring Boot controller-service-repository 계층.
- DTO와 Entity 분리.
- JPA relationship.
- 채팅에서 WebSocket과 polling의 차이.
- Naver News API 연동.
- JWT 기반 인증.
- 프론트엔드와 백엔드 테스트 전략.
