# Jira Workflow

## Board Statuses
- Backlog
- Selected
- In Progress
- Code Review
- Done

## Ticket Types
- Epic: large product area.
- Story: user-facing capability.
- Task: implementation or documentation work.
- Bug: incorrect behavior.
- Spike: investigation before implementation.

## Done Definition
- Acceptance criteria are satisfied.
- Relevant tests or manual checks are completed.
- Documentation is updated when behavior or architecture changes.
- AI implementation notes are added when new concepts are introduced.
- Code is committed and pushed.

## Branch Naming
```text
feature/CKA-123-short-description
bugfix/CKA-456-short-description
docs/CKA-789-short-description
```
